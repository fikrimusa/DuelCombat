# DuelCombat Game

A third-person combat game where players engage in duels with various attacks and abilities.

## Controls

- **W**: Move forward
- **A**: Move left
- **S**: Move backward
- **D**: Move right
- **Mouse movement**: Look around
- **Space Bar**: Jump
- **Right Click (Mouse)**: Basic attack
- **Left Click (Mouse)**: Heavy attack
